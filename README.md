 📌 Changelog Timeline Component

A clean and modern **Changelog Timeline UI** built using **HTML & CSS**, inspired by roadmap-style designs.
It displays updates in a vertical timeline with **dates on the left**, a **centered line with dots**, and **content on the right**.



 🚀 Features

* 📍 Centered vertical timeline
* 📅 Dates aligned on the left side
* ⚫ Perfectly aligned timeline dots
* 📝 Content on the right side
* 📱 Fully responsive (mobile-friendly)
* 🎨 Clean card-based UI design
* 🧩 Easy to customize and reuse



 🛠️ Technologies Used

* HTML5
* CSS3 (Grid + Flexbox)



 📷 Preview

This component visually represents a changelog in a structured timeline format similar to modern UI designs.
